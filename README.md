# Smart Ping Pong Agent

## Project Overview
This project is an autonomous agent capable of playing Ping Pong by tracking the ball and defeating an opponent. It implements a **Model-Based Reflex Agent** to calculate optimal paddle positioning in real-time.

## Environment & Agent Characteristics
* **Agent Type:** Model-Based Reflex Agent
* **Sensors:** Ball Position (x, y), Ball Velocity, Paddle position.
* **Actuators:** Vertical movement of the paddle.
* **Environment Properties:** Fully Observable, Deterministic, Sequential, Dynamic, Multi-Agent.
